# ⏳ Chronos

A minimalist Time Tracking & Reflection Engine built with Python and Streamlit.

Chronos helps users track how they spend their time, visualize productivity, and generate a daily journal from tracked activities.

---

## ✨ Features

- ⏱️ One-click Start/Stop timer
- 📊 Daily analytics dashboard
- 🥧 Interactive charts with Plotly
- 📝 Automatic journal generation
- 📂 Custom categories
- 🎯 Daily productivity goals
- 📅 Session history
- ⚙️ User settings

---

## 🛠 Tech Stack

- Python
- Streamlit
- Pandas

## 🚀 Installation

```bash
git clone https://github.com/Jenish-sr71/Chronos_pro.git

cd Chronos_pro

python -m venv venv

# Windows
.\venv\Scripts\activate

pip install -r requirements.txt

streamlit run app.py
```
